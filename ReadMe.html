クイズゲーム（ベータ版）ver1.01

1.はじめに

本書は、クイズゲーム（ベータ版）の使い方やベータ版でできないこと、今後追加する予定の機能等を記載します。

2.対応プラットフォーム

Google Chrome　がインストールされたWindows10(32ビット版/64ビット版) 　 Windows8.1(32ビット版/64ビット版)IEは使えません。

3.ベータ版でできないこと、付け足したいところ

メニュー(ユーザー名がかかれているところ）が使えない。

問題を作れない

ローカルストレージに保存しているため、他機器で利用できない。（Firebase対応予定)

カウントダウンが使えない。

ログアウトしてもランクとポイント等の表示がリセットされない。

SNSに進行度を上げる。 　仲間の進行度を見る

音声再生機能（音楽聞いて曲名答えるなど）

UIの大幅変更（見た目があまり良くないため）

一部のページが適当なのは許して。 以上
